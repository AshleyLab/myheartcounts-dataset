# Install from the repo root (torch is pinned to +cu126, so every use needs the cu126 index):
#   IDX="--extra-index-url https://download.pytorch.org/whl/cu126"
#   CON="-c scripts/paper_results/constraints.txt"
#   CPU methods (bit-exact):  pip install -e ".[downstream]" $CON $IDX
#   Full env (all 8 methods): pip install -e ".[all]"        $CON $IDX
#     then the two steps docs/install.md documents (their pins below are inert guards):
#       toto: pip install --no-deps toto-ts==0.2.0 ; pip install -e ".[toto_dep]" $CON $IDX
#       wbm : source-build mamba-ssm + causal-conv1d (the PyPI sdist omits csrc/ — see install.md)
#   (CPU-only and don't want the CUDA torch? drop torch from the resolve / use a CPU wheel.)
#
# CPU bit-exactness holds ONLY on: linux x86_64 (manylinux2014) · CPython 3.10 · pip (NOT conda).
# OpenBLAS is bundled in the numpy/scipy wheels, so pinning those pins the BLAS by proxy
# (numpy 1.26.4 -> OpenBLAS 0.3.23.dev ; scipy 1.15.3 -> 0.3.28). conda/MKL/other arch = tolerance.
